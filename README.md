# Análise e Projetos de Sisemas Orientado a Objetos
### 1. O que é uma Variável?
De forma simples, uma variável é um espaço na memória do computador onde guardamos um dado que pode mudar durante a execução do programa. Pense nela como uma caixa etiquetada.
A Etiqueta: É o nome da variável (ex: preco_do_cafe).
O Conteúdo: É o valor guardado (ex: 5.50).
### 2. O que é uma Classe?
A Classe é o projeto, a planta baixa ou o "molde". Ela não é o objeto em si, mas sim a definição do que ele deve ter e como deve se comportar.
Exemplo: Se fôssemos criar um sistema para uma fabricante de carros, a classe seria o projeto do engenheiro que descreve que todo carro tem cor, modelo e a capacidade de acelerar.
### 3. O que é um Objeto?
O Objeto é a "instância" da classe. É quando você pega o molde (classe) e cria algo real e concreto a partir dele.
Exemplo: Se a classe é "Carro", o objeto é o Fusca azul de placa ABC-123 que está na sua garagem. Ele existe fisicamente na memória do computador.
## 4. O que são Atributos de uma classe?
Atributos são as características ou dados de uma classe. Eles definem o "estado" do objeto. Se a classe for um substantivo, os atributos são os adjetivos.
Exemplos de atributos para a classe Carro:
Cor: Vermelho
Marca: Ferrari
Velocidade Máxima: 320 km/h
### 5. O que são Métodos de uma classe?
Métodos são as ações que o objeto pode realizar. Eles definem o comportamento. Se os atributos são adjetivos, os métodos são os verbos.
Exemplos de métodos para a classe Carro:
acelerar()
frear()
trocarMarcha()
#### 6. Qual a relação entre Classe e Objeto?
A classe diz como deve ser, e o objeto é a execução dessa definição. Você pode usar uma única classe para criar milhares de objetos diferentes (vários bolos com sabores diferentes usando a mesma forma).

#### Resumo rápido: 
Uma classe (molde) possui atributos (características) e métodos (ações). Quando damos vida a esse molde, criamos um objeto, cujos dados específicos são armazenados em variáveis.
